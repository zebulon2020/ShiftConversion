移位变换算法的实现

加解密原理：

$ c=E_{k}(m)  \equiv m+k \pmod{26} ,0\le m,k\le 25 $  
$ m=D_{k}(c)  \equiv c-k \pmod{26} ,0\le c, k\le 25 $
